https://real-amber-eight.vercel.app/

This project is a full-stack web application that includes a dynamic landing page and an admin panel for managing projects, clients, contact form submissions, and newsletter subscribers.

✨ Features
⭐ Landing Page

Displays all Projects (image, name, description)

Shows Happy Clients testimonials

Fully functional Contact Form (Name, Email, Mobile, City)

Newsletter Subscription option

All data fetched dynamically from backend APIs

⭐ Admin Panel

Manage Projects (Add/Edit/Delete)

Manage Clients (Add/Edit/Delete)

View Contact Form Submissions

View Newsletter Subscribers
